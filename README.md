# 🚗 LA Auto Center

Site de vendas de carros de luxo feito com HTML, CSS e JavaScript.

## 📦 Estrutura

- Home
- Catálogo
- Sobre
- Contato

## 🚀 Como usar

Abra o arquivo `index.html` no navegador.

Ou publique no GitHub Pages:
1. Suba os arquivos no GitHub
2. Vá em Settings → Pages
3. Selecione branch `main`

## 💎 Tecnologias

- HTML5
- CSS3
- JavaScript
